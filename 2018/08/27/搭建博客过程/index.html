前言

​ 一直点下一步即可,安装完成后 cmd 在命令提示窗口中输入 node -v 如果出现版本信息证明安装成功

将本地博客上传到github

1. 登录github 创建一个新项目

项目名为: 你github的用户名.github.io

2. 进入到你创建的项目

点击Setting 拉到下面找到GitHub Pages 点击Choose a theme

随便选一个主题 点击Select theme

Repository name 中的内容不要修改

然后点击Set up templates即可

3. 访问一下你的连接

输入地址: 用户名.github.io

如果能看到你刚刚选择的主题 说明没有问题

这样一个简单的个人博客就搭建成功了,这只是最基础的, 对于主题github中就有很多开源的项目, 可以自行进行选择,如果你不喜欢github提供的域名,可以使用自己的域名,接下来讲一下如何购买一个域名做绑定

域名绑定

这里我用的是腾讯的域名

1.登录腾讯云进入到控制台

上方有个域名注册, 进入后点击注册域名, 然后进行购买即可, 购买完成后返回控制台域名注册页面, 根据提示进行实名认证, 提交后会进行审核, 请耐心等待, 审核成功会发短信通知你

之后点击当前域名的解析操作 添加记录

这是免费的 你也可以通过腾讯云右上角的云产品找到云解析然后后买云解析套餐 个人的话一年好像是300多

主机记录可以任意选择 记录类型选择CNAME 记录值填你github的名称.github.io. 注意后面一定要多加一个.

点击保存即可

然后到你博客 根目录/source 目录下创建一个新文件CNAME 吧后缀名删掉
在里面写上你刚刚配置的路径，比如我的是muzixiaozhe.cn，就直接在CNAME文件中写上这个地址就好了

然后再次执行
